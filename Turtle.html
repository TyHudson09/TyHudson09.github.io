Turtle Project

About the Project

This project is about creating a graphical scene using Python’s Turtle graphics module.I used the provided building block functions (draw_rectangle, draw_square, draw_triangle, draw_circle) to create functions that draw composite shapes such as a house, clouds, sidewalks, etc. Position functions such as base_x, base_y, provide coordinates to position the building fuctions to create the image

Sample Code

This function creates a cloud shape using multiple overlapping circles. The parameters allow for flexible positioning and scaling:

• t: The turtle graphics object for drawing
• base_x, base_y: Coordinates moved from base for the cloud
• radius: Radius of the cloud
• Color: Color of the cloud

def draw_cloud(t, base_x, base_y, radius=25, color="white"):
"""Draw a cloud using multiple circles at specified base position with customizable radius and color"""
# Define relative positions for cloud circles to form a cloud shape
positions = [
(base_x, base_y), # Center circle
(base_x - 25, base_y + 5), # Left circle
(base_x + 25, base_y + 10), # Right circle
(base_x + 50, base_y) # Far right circle
]
for x, y in positions: # Draw a circle at each position
jump_to(t, x, y)
t.pencolor("white")
draw_circle(t, radius, fill_color=color)
