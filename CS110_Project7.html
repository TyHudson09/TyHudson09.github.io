# prompt: Write a code plotting a heat map of the top 5 ITEM TYPE and their RETAIL SALES

# Reshape the data for the heatmap
# The heatmap function expects the data to be structured
# so that the categories you want on the y-axis (Item Type)
# are the index and the categories on the x-axis (in this case, the single metric 'Retail Sales')
# are the columns.
heatmap_data = top_5_items.set_index('ITEM TYPE')

# Create a heatmap
# Set the size of the figure for better readability
plt.figure(figsize=(8, 4))

# Generate the heatmap using the seaborn library
sns.heatmap(
    heatmap_data,          # The data to plot (top 5 items with Item Type as index)
    annot=True,            # Display the data values in each cell of the heatmap
    fmt='.2f',             # Format the annotations to two decimal places
    cmap='YlGnBu',         # Set the color map (Yellow-Green-Blue) to visualize intensity
    cbar_kws={'label': 'Retail Sales'} # Add a color bar and label it
)

# Add a title to the heatmap
plt.title('Heatmap of Top 5 Item Types by Retail Sales')

# Label the x-axis (in this case, it represents the single metric 'Retail Sales')
plt.xlabel('Metric')

# Label the y-axis (these are the Item Types)
plt.ylabel('Item Type')

# Adjust plot to prevent labels from overlapping
plt.tight_layout()

# Display the plot
plt.show()

Based on the provided data and visualizations, I can effectively answer the research question: What item types were the top 5 in retail sales. The bar chart and heatmap clearly illustrate the top 5 item types based on their aggregated retail sales. From these visualizations, it's evident that liquor is the highest-selling item type, followed by Wine and Beer. The difference in retail sales between the top item and the others appears to be moderate for the first three then a large drop on 4 and 5, suggesting a dominant performers in the market.

While the visualizations provide a clear answer to the primary question, this analysis has certain limitations. It focuses solely on the overall retail sales volume and doesn't account for profitability, which would require data on the cost of goods sold. Additionally, this analysis doesn't consider the time dimension; examining sales trends over time could reveal seasonality or growth patterns for these top items. Further investigation could involve analyzing the impact of wholesale sales on retail performance or exploring geographical variations in sales for these popular item types.
